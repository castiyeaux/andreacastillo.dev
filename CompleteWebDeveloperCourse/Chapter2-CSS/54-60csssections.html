Relative: Positioned relative to where it would otherwise be.

-- Left: Looks like it's moved right, but really more like there is a static (+)Xpx margin on the left side. A (-)Xpx margin on the left side will make it look like it moved left.

-- Top: Looks like it's moved down, but really more like there is a static (+)Xpx margin on the top side. A (-)Xpx margin on the top side will make it look like it moved up.

Other Types of Positioning

Positioning Ex. 7

Positioning Ex. 8


Ex. 8 is still on the page, just behind Ex. 7 (made width bigger to see).

Absolute: Takes the object to be positioned out of the flow of the page. i.e. all other elements positioned themselves as if the absolute element wasn't there at all.

- Really useful if you want something to be on top of everything else and for everything else to ignore it.

- Can also adjust left/top margins, like with relative.

- Will move when scrolling page.

----

Positioning Ex. 9 to be set at 'position: fixed;'.

Positioning Ex. 10

Fixed: Also removes object from the flow of the page.

- May need to specify 'left' and 'top' px for it to actually show.

- Can also adjust left/top margins, like with relative and absolute.

- Will NOT move when scrolling page.
